# SnakeGame

2D Python Snake game made using Pygame.

![](snake_preview.gif)

## Installation

Python 3.8.2 (must be 3.6.5 or above):
```python
https://www.python.org/
```
Libraries:
```pygame
pip install pygame
```

## Usage
At the terminal in the root of the project directory type:
```run
python snake_game.py
```
Controls:
- w - up
- s - down
- a - left
- d - right
